üóìÔ∏è September, 2021: pyCATHY is born¬∂

I started to work on pyCATHY in order to learn how to use the CATHY code originally developed in Fortran (first version developed by A. Binley dated from ??). The pyCATHY package consists in a python wrapper easing the numerical modelling of hydrology by developing object oriented class and modules to interact with core CATHY functions.

import CATHY

simu_hydro = CATHY()

The aim is to develop a version-agnostic wrapper allowing to work on the differences branches of CATHY. In particular we‚Äôd like to integrate Data Assimilation capabilities and the plant model to the wrapper in order to mimick a soil-plant-interaction experiment run in the laboratory using geoelectrical ERT and MALM methods.
