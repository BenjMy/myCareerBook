    • Code dev repository

    • How to cite: Zenodo DOI

    • Documentation

    • Articles from this work: in progress

    • How to cite: Zenodo DOI

    • Documentation

    • Articles from this work: in progress

  • pyCATHY: open-source python package wrapper aiming at coupling hydrological forward modelling with geophysical data by Camporese et al. .

    • Code repository
